## 💡 已知限制

### 当前版本限制

1. **流式响应**: 仅框架实现，实际功能待完善
2. **数据转换**: 未实现，当前仅支持原生格式
3. **OAuth 认证**: Gemini/Kiro/Qwen 未实现
4. **系统提示词**: 仅配置加载，未集成到请求中
5. **错误重试**: 未实现自动重试机制
6. **日志系统**: 基础日志，缺少结构化日志

### 性能优化机会

1. **连接池**: HTTP 客户端可优化连接复用
2. **缓存**: 可添加模型列表缓存
3. **并发**: 可优化并发请求处理
4. **内存**: 可优化大对象的内存分配
